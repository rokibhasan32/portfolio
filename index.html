Research

Enhancing Sports Performance with Big Data and Machine Learning-Powered Predictive Analytics

Status: Ongoing Research

Developed an end-to-end interpretable AI framework for tactical analysis of football matches using big data and XAI techniques. Implemented a custom YOLOv8 object detection model combined with DeepSORT tracking to extract player trajectories from full-match broadcast videos.

Applied camera motion compensation and top-down projection to calculate spatial-temporal features such as player speed, distance covered, and team formations. Integrated an XAI layer using Grad-CAM and SHAP to generate transparent, human-readable explanations of tactical insights.

Achieved strong performance metrics: mAP@0.5 = 0.677, and high accuracy for key classes (Player AP: 0.990; Referee AP: 0.995; Goalkeeper AP: 0.871). Enabled actionable insights for coaching staff, including event recognition (e.g., possession turnovers) and contextual performance metrics.

🤖
